hoja de vida

Una PC (del inglés Personal Computer, que significa "computadora personal") es un dispositivo electrónico diseñado para ser utilizado por una persona a la vez. Está diseñada para realizar una variedad de tareas, como procesamiento de datos, navegación web, edición de documentos, juegos, programación y más.

elementos

Hardware (partes físicas): CPU (Procesador): El "cerebro" de la computadora, encargado de ejecutar instrucciones. RAM (Memoria): Almacena datos temporalmente para un acceso rápido. Disco duro (HDD/SSD): Almacena el sistema operativo, programas y archivos. Placa madre (Motherboard): Conecta todos los componentes. Tarjeta gráfica (GPU): Procesa imágenes y videos (esencial para juegos y diseño). Fuente de poder: Proporciona energía a los componentes. Periféricos: Teclado, mouse, monitor, etc.
